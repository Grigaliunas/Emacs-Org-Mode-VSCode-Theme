# Emacs Org Mode (VS Code)


## Install (manual)
1. Copy this folder into your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
   - **macOS**: `~/.vscode/extensions/`
   - **Linux**: `~/.vscode/extensions/`

   Example final path:
   `.../.vscode/extensions/emacs-org-mode-theme/`

2. Restart VS Code.
3. `Preferences: Color Theme` â†’ **Emacs Org Mode**

## Optional editor font
Theme files cannot enforce fonts; set this in VS Code settings if you want matching typography:

```json
{
  "editor.fontFamily": "Courier New, monospace",
  "editor.fontLigatures": false
}
```
